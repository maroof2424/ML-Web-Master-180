# Week 15 – OpenCV

## Goal
Learn image processing, face detection, webcam capture, and build a Streamlit + OpenCV live filter app.

## Daily Plan

### Day 1 → Intro to OpenCV
- Install OpenCV
- Read & display images
- Convert to grayscale
- Resize, crop, draw shapes

### Day 2 → Basic Operations
- Image rotation, flipping
- Color channels (RGB/BGR)
- Blur, threshold

### Day 3 → Filters & Edge Detection
- Gaussian blur
- Canny edge detection
- Sobel filters

### Day 4 → Face Detection
- Haarcascade face detection
- Draw bounding boxes around faces

### Day 5 → Webcam Processing
- Open webcam
- Apply filters in real-time
- Edge filter, sketch filter, grayscale

### Day 6 → Mini Project
- Streamlit + OpenCV Live webcam filter app

### Day 7 → Review
- Compare filters
- Final touches & deployment
