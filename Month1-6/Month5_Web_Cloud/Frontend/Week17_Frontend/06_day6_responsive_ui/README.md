## Overview
This project demonstrates how to create a mobile-friendly, responsive user interface using a combination of **Bootstrap**, **TailwindCSS**, and **custom CSS**. The design adapts to different screen sizes (mobile, tablet, desktop) and includes interactive elements.

## Features
- **Navbar:** Responsive navigation bar using Bootstrap classes.
- **Hero Section:** Gradient background with title, description, and call-to-action button.
- **Cards Section:** Responsive feature cards with hover effect using Tailwind + custom CSS.
- **Mobile-friendly Layout:** Fully responsive design that works on any screen size.

## Technologies Used
- HTML5
- Bootstrap 5 (CDN)
- TailwindCSS (CDN)
- Custom CSS for hover effects and typography

## How to Run
1. Download the `responsive.html` file.
2. Open the file in any modern web browser.
3. Resize the browser window to see the responsive behavior.
