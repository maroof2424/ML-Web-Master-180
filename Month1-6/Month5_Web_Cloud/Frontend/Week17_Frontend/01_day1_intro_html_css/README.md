## HTML
- Structure: `<html> <head> <body>`
- Headings: `<h1>` to `<h6>`
- Paragraphs: `<p>`
- Links: `<a href="#">Link</a>`
- Images: `<img src="..." alt="...">`
- Lists: `<ul>` / `<ol>` with `<li>`

## CSS
- Inline, Internal, External
- Color, font-size, background
- Margin & Padding
- Basic selectors: element, class, id
