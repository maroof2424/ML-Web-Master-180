# **Day 1 – Power BI Basics & Data Loading**

## **Objective**

Power BI ka interface samajhna aur first dataset load karna.

> Goal: “Data → Power BI → Ready for analysis”

---

## **1️⃣ Why Power BI**

* Excel se advanced visualization & modeling
* Interactive dashboards
* Business Intelligence reporting
* Large dataset handling

> Think: Power BI = Excel + SQL + Charts + Dashboard in one tool

---

## **2️⃣ Power BI Components**

| Component          | Purpose                              |
| ------------------ | ------------------------------------ |
| Power BI Desktop   | Build reports & dashboards           |
| Power Query Editor | Data cleaning & transformation       |
| Data Model         | Relationships between tables         |
| Reports            | Visual representation (Charts, KPIs) |
| Power BI Service   | Publish & share dashboards           |

---

## **3️⃣ Data Sources**

* Excel, CSV, SQL Server, Web, API
* Load multiple tables
* Power Query for preprocessing

---

## **4️⃣ Hands-On: Load Demo Dataset**

1. Open Power BI Desktop → `Get Data → Excel → select file`
2. Select `sales_data.xlsx`
3. Preview table → `Load`

### ✅ Pro Tips

* Check column data types (Text, Number, Date)
* Rename messy columns
* Avoid blank headers

---

## **5️⃣ Data View vs Report View vs Model View**

| View        | Purpose                    |
| ----------- | -------------------------- |
| Data View   | See table rows & columns   |
| Report View | Build charts & dashboards  |
| Model View  | Define table relationships |

---

## **6️⃣ Real-World Use Case**

* Load monthly sales dataset
* See raw numbers in **Data View**
* Switch to **Report View** → ready to visualize KPIs

---

## **7️⃣ Best Practices**

* Always **rename columns**
* Correct **data types** before modeling
* Keep dataset **clean & consistent**
